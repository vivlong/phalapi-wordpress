# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- PHP 8.0+ support with strict types
- Constructor property promotion
- Union types
- Match expressions
- PSR-12 coding standard
- PHP-CS-Fixer, PHPCS, PHPStan, and PHPUnit configurations
- Improved code documentation
- Test suite structure

### Changed
- Upgraded GuzzleHttp to ^7.0
- Modernized codebase to PHP 8.0+ standards

## [1.0.0] - 2024-01-01

### Added
- Initial release
- WordPress REST API client
- Support for posts, pages, media, categories, tags, users, comments
- JWT and Basic authentication
- File upload support
